# Vue.js Frontend

This is the Vue.js frontend for the Full Stack Application with FastAPI backend.

## Project Setup

```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run serve
```

### Compiles and minifies for production
```bash
npm run build
```

### Lints and fixes files
```bash
npm run lint
```

## Features

- Vue.js 3 with Composition API
- Axios for API calls
- Responsive design
- Integrated with FastAPI backend via API endpoints
